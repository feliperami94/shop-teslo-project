# Teslo API
1. Clone project
2. ```npm i```
3. Clone  ```.env.template``` and rename it as ```.env```
4. Change environment variables
5. Build the DB
```
docker-compose up -d
```
6. Run ```npm run start:dev```

7. Execute Seed
```
http://localhost:3000/api/seed
```
